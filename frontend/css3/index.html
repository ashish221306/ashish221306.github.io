CSS3 Viewport width and viewport height

CSS3 introduced some new units for height and font size. These units are vw, vh and vmin. Viewport width or Viewport height are related to device viewport, i.e visible screen of device to user.

  1. 1vw = 1% of viewport width.
  2. 100vw = 100% of viewport width.
  3. 1vh = 1% of viewport height.
  4. 1vmin = 1% vw or vh, whichever is smaller.
  5. 1vmax = 1% vw or vh, whichever is larger.

Note !

  1. 1vw= 13.66px if screen resolution is 1366*768
  2. 1vw= 14.4px if screen resolution is 1440*900
  3. 1vw= 3.6px if screen resolution is 360*640
  4. 1vh= 6.4px if screen resolution is 360*640

CSS3 Vendor Specific prefixes

To Avoid cross browser compatibility of css3 based properties in latest and old browsers, use following vendor specific prefix for css3 properties.

  1. -webkit-
  2. -moz-
  3. -ms-
  4. -o-

Example

    <style>
  .logo{
  border-radius:10px;            /*For all Latest Browsers*/
  -webkit-border-radius:10px;    /*For Chrome, Safari Android & IOS  */
  -moz-border-radius:10px;       /*For Mozilla Based Browsers*/
  -ms-border-radius:10px;        /*For IE 9 and above*/
  -o-border-radius:10px;         /*For Opera 11 and above browsers*/
  }  
  </style>
